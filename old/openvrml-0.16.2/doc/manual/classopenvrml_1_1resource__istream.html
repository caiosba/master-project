openvrml::resource_istream

openvrml::resource_istream Class Reference

List of all members.

Detailed Description

An abstract input stream for network resources.

resource_istream inherits std::istream, adding functions to get the URI and the MIME content type associated with the stream. Users of the library must provide an implementation of this class, to be returned from openvrml::browser::do_get_resource.

See also:
openvrml::browser::do_get_resource(const std::string &)
